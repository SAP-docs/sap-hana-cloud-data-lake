# TAN Function for Data Lake Relational Engine \(SAP HANA DB-Managed\)

Returns the tangent of a number.



```
TAN ( <numeric-expression> );
```



<a name="loioe2c70619ddcb41a58cd65f42dc1bab7f__section_o3l_1kv_vrb"/>

## Parameters


<dl>
<dt><b>

*<numeric-expression\>*

</b></dt>
<dd>

An angle, in radians.



</dd>
</dl>
