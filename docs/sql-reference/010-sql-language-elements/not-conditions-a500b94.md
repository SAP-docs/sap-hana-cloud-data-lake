# NOT Conditions

The NOT condition can be either TRUE, FALSE, or UNKNOWN.



The syntax for `NOT` conditions is:

```
NOT <condition1>
```

The `NOT` condition is TRUE if *<condition1\>* is FALSE, FALSE if *<condition1\>* is TRUE, and UNKNOWN if *<condition1\>* is UNKNOWN.
