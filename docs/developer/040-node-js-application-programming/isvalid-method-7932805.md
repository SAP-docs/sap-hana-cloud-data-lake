# isValid\(\) Method

Checks whether the statement is valid.



```
statement.isValid()
```



<a name="loio7932805092d1491babc93f6e2964b1b5__section_alx_x3v_x1b"/>

## Returns

Returns true if valid, and false otherwise \(the statement has been dropped or is being dropped\).
