# isClosed\(\) Method

Checks if the result set is closed.



```
resultset.isClosed()
```



<a name="loio1bf45fb89fa849b393f12bf68c9fbc96__section_alx_x3v_x1b"/>

## Returns

This function returns true if the result set is closed and false otherwise.
